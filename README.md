# 🚦 Smart Traffic Management System Using YOLOv11 and GBML-DQN

This repository presents a **Smart Traffic Management System** that uses **Artificial Intelligence and Computer Vision** to improve traffic signal management at road intersections. The project combines **YOLOv11** for real-time vehicle detection with a **GBML-DQN (Gradient Boosted Machine Learning - Deep Q Network)** model to make intelligent decisions about traffic signal timing based on current traffic conditions.

Instead of relying on fixed signal durations, the system analyzes live traffic density and dynamically adjusts signal timings to reduce congestion, decrease vehicle waiting time, and improve overall traffic flow. This approach demonstrates how AI-driven technologies can contribute to smarter and more efficient urban transportation systems.

## 🚀 Features

* Real-time vehicle detection using **YOLOv11**
* AI-based traffic signal optimization
* Dynamic signal timing based on traffic density
* Intelligent decision-making using **GBML-DQN**
* Designed to reduce congestion and improve road efficiency

## 🎯 Objective

The main objective of this project is to demonstrate how Artificial Intelligence can be applied to modern traffic management systems. By integrating computer vision with reinforcement learning, the system aims to create a more adaptive and efficient method for controlling traffic signals, ultimately helping to reduce congestion and improve the overall transportation experience.
